Automating the Sync (Cron Job)

Since your worker is now stable, you can automate it.

Log into cPanel.

Search for "Cron Jobs".

Set the timing to "Once Per Hour" (or use 0 * * * *).

Enter this command (adjusting the path to your server):

/usr/local/bin/php /home/kellgradeschool/www/www/kgs2026/ac/kgs-core/workers/refresh-drive-cache.php > /dev/null 2>&1

REMEMBER TO CHANGE THE PATH WHEN WE MOVE THE SITE TO ITS DESTINATION
